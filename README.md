## 🚀 How to use

To run the following command on NeoVim.

```
:RunIt
```

## ⬇️  Install

- dein.vim

```
[[plugins]]
repo = "https://github.com/Comamoca/runit.nvim"
on_cmd = ["RunIt"]
```

- dein.vim(toml)

```
call("dein#add", "https://github.com/Comamoca/runit.nvim")
```

## ⛏️   Development

Clone this repository and edit `./lua/runit.lua`.
