## 🚀 使い方

NeoVimで以下のコマンドを実行してください。

```
:RunIt
```
## ⬇️  Install

- dein.vim

```
[[plugins]]
repo = "https://github.com/Comamoca/runit.nvim"
on_cmd = ["RunIt"]
```

- dein.vim(toml)

```
call("dein#add", "https://github.com/Comamoca/runit.nvim")
```

## ⛏️   開発

このリポジトリをクローンして`./lua/runit.lua`を編集してください。
